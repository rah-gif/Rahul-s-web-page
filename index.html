Project Two

This semester, I embarked on an exciting project involving the development of an Arduino-based ultrasonic system. This innovative project utilizes ultrasonic sensors to measure distances accurately and can be applied in various fields, such as robotics, automotive, and home automation. The primary objective of this project is to design a system that can detect obstacles and provide real-time feedback based on distance measurements. The ultrasonic sensor emits sound waves that bounce off nearby objects and return to the sensor, allowing the system to calculate the distance to the object based on the time taken for the waves to return.

Project Three

In my recent project, I utilized Tinkercad to design and simulate an Arduino-based system that integrates LEDs and a 74HC32 logic gate. This project aims to demonstrate the application of logic gates in controlling electronic components and enhancing the functionality of basic circuits. The 74HC32 is a quad two-input OR gate, which allows for the combination of multiple input signals to control the output state of the LEDs. By programming the Arduino, I created a system where the LEDs would light up based on specific logical conditions defined by the inputs to the 74HC32 gate.
